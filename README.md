Run `npm start` for a dev server. Navigate to `http://localhost:58967/angular-playground/`. The application will automatically reload if you change any of the source files.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Running unit tests

Run `npm test` to execute the unit tests via Vitest.
